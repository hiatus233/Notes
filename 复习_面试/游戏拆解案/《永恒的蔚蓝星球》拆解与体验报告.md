# 简介
《星球》是一款核心玩法为局内轻度肉鸽+塔防，通过金币召唤英雄，相同星等的英雄随机合成更高等级的英雄，消灭敌人来通关；局外长线上引入一套常规rpg系统的休闲策略类游戏。


|      |                   |
| ---- | ----------------- |
| 平台   | 移动端 微信/字节/TapTap等 |
| 游戏类型 | 休闲策略              |
| 玩法   | 肉鸽、塔防+rpg         |
| 美术风格 | 卡通可爱              |
| 付费模式 | 主IAP+轻量IAA        |
核心循环：闯关获取资源->提升战力->解锁新关卡->获取高阶资源

# 战斗
## 概述
《星球》中的战斗玩法融合了传统的合成塔防和轻量肉鸽随机构筑的玩法机制设计，局内唯一的资源是银币，通过银币可以召唤英雄/给已召唤英雄添加词条；相同星级的英雄可以随机合成出更高一级的英雄(最高为四级，从一级起算)，合成到四级的时候会固定赠送一个合成出的四级英雄的词条（跟消耗银币添加词条一样是三选一）。
## 英雄
《星球》中的英雄可以类比成传统塔防游戏中的防御塔，在局内玩家可以通过消耗银币对其进行升级，具体的合成设计规则如概述中说的一样，两个相同等级的英雄能够随机合成一个新的更高一级英雄。

## 编队
在进入关卡战斗前，玩家需要选择自己的编队，多样的职业选择会给队伍带来属性上的加成。（这里卡史诗及以上品质的英雄也是为了提醒玩家前期不要一直培养低品质的英雄卡。）
![[file-20260312011513543.png]]
一共有三队的编队容量。编队的设计主要是便于玩家针对不同的情况选出不同的阵容来过关。

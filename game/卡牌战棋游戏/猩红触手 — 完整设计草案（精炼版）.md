# 猩红触手 — 完整设计草案（精炼版）

## 核心概念

大型跨层 Boss，**横跨三层**：上/下层布置触手小怪（阶段一），当小怪被清光 Boss 降临中层（阶段二），最终进入狂暴/地形破坏阶段（阶段三）。设计目标：制造“多层压力 + 地形威胁 + 强迫优先级选择（先清小怪还是直接伤Boss）”。

---

## 视觉/体验要点

- 视觉：巨型触手体外露，上一层和下一层可见“触手根基”，中层Boss出现时镜头聚焦并有落地动画。
    
- UI：每个阶段开始播放简短提示条（“触手兵群出现”，“主触手现身”）。意图 UI 与攻击范围必须清晰（使用现有 intentUIImage / ShowDetailedPrediction）。
    

---

## 阶段细节（行为、AI、数值示例）

> 数值为示例，调平衡用变量化配置（在SO或Inspector上）。

### 阶段一：残肢触手（分散在上下两层）

- **触手种类**（小怪，简单AI）：
    
    1. **射手触手**：远程攻击，意图为远程单体/散射；血量低，优先被清。（HP 6, STR 4, range 4）
        
    2. **牧师触手**：每 N 回合给己方一个小护盾或治疗（优先回血友军），意图为支持（HP 8, 能治疗或增甲）。
        
    3. **战士触手**：近战、冲锋，会推动并破坏地块（攻击带地形伤害）。（HP 10, 会对目标脚下地块造成 1 点耐久伤害）
        
- **刷怪/位置**：在起始房间的上层与下层分别生成若干（例如：上层 2 射手 + 1 牧师，下层 2 战士），位置在 MapTemplate 的 enemySpawnArea 指定层坐标。
    
- **设计目标**：逼玩家分散注意力，若玩家忽视，触手会累计对该层地形的破坏度（地形破坏带来长期压力）。
    

### 阶段二：Boss 降临（中层，80% 初始血）

- **触发条件**：所有阶段一小怪被消灭后（或触发计时器达到 X 回合也可强制触发）。
    
- **Boss 基本属性**（示例）：HP 120（降到 80%/80 起始）、STR 12、MoveRange 2、AttackRange 2。
    
- **三种主要技能（随机/加权选择）**：
    
    1. **拖拽（Grab & Slam）**：选定一名玩家，从其所在层任意位置拖到 boss 前方并立即一次重击（即使跨层也可通过传送样式实现）。效果：强制位移 + 重击（高伤害）。（意图：显示目标箭头 + 拖拽路径）
        
    2. **周围摧毁（Shred Ground）**：摧毁自身周围一圈地块（对地块造成大量耐久伤害或直接 Annihilate），并对处于这些地块的单位造成伤害/坠落效果。意图：范围破坏（ActionRange） + 地块损害预览（ShowTileDamagePreview）。
        
    3. **酸雾/溅射（Acid Spray / Corrosive Pool）**：在目标区域喷吐酸液，导致地块转化为酸池（与 CorrosiveAcidStrategy 对应），对被覆盖玩家造成持续伤害/移速惩罚。意图：范围标记 + 地形预览（ShowTerrainChangePreview）。
        
- **战场控制**：Boss 能召回阶段一触手若干（召唤技能），优先策略为“保持场上压力”。
    

### 阶段三：狂暴 / 终结（低血线触发，≤50%）

- **触发**：HP ≤ 50% 或在若干回合内未被削弱特定条件（例如未摧毁所有触手根部）。
    
- **行为改变**：
    
    - 攻击频率提升（或每回合两次动作）。
        
    - **层间冲击**：对上下层同时投放炸裂攻击，随机/目标化破坏多格地形并造成大范围伤害（迫使玩家利用房间纵深/上层保护）。
        
    - 可能施放“地形翻转/重生阻断”技能（令被摧毁地块在短时间内不能重生或转为致命地形），短期提高破坏度压力。
        
- **目的**：让战斗到达高潮，迫使玩家做出保守或冒险决策（保命 vs 高伤）。

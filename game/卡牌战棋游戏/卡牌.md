### 移动类
参考陷阵之志和将军对决。
对玩家/敌人产生位移作用的牌都称为移动牌，这些牌能使Entity与地块上的碰撞物产生交互（包括但不限于碰撞，坠落，燃烧等）

| 名称  | 卡牌消耗 | 卡牌描述         | 卡牌稀有度 |
| --- | ---- | ------------ | ----- |
| 推动  | 0    | 往后推动指定单位2格距离 |       |
| 传送  | 1    | 传送到上层或下层     |       |
|     |      |              |       |

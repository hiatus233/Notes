# 花果山
花果山通过提供属性成长的部将或【神通】，来使具有【协战】特性的部将触发【协战】，进而使花果山的所有部将获得持续的属性成长。
# 天庭
天庭通过【登场】体系，通过经济核心王灵官在中期寻找阵容组件，并通过辅助核心斗姆元君对三太子进行属性提升。天庭也可以以三太子、李天王，为核心打造【护盾】阵容。
# 迦楼罗
迦楼罗非常考验操作者的决策和操作能力，能否拿到【飞行】流派的关键卡【大龙神】、【大羽神】或【抢攻】流派的关键卡【幻羽】，并且如何快速提高关键卡牌的属性是取胜的关键。
# 大唐
大唐拥有最多的部将数量，可提供最多的辅助能力，也最灵活多变。可通过唐王和持续获得经济优势的部将，打造跨阵营的阵容。
# 狮驼岭
狮驼岭擅于提升商店部将属性，然后由阵中部将吞噬商店部将来获得属性成长。
# 地府
地府属于高攻低血的阵营，通过冥府女帝、冤魂、鬼魂神通、夜游神增强地府部将的攻击力；再在战斗中不断召喚地府部将，以获得战场上的优势
# 阿修罗
阿修罗是卖血的阵营，战争巨象在大圣受到伤害时可以为友方部将提供大量属性。通过部将的购买、登场、商店刷新、等机制扣除大圣血量，再配合鬼将将伤害降为0，达成完美的循环。
# 灵山
灵山通过叠加【灵力】来增加法术伤害，往往和【残念】进行搭配。通过地藏王菩萨提前触发已方【残念】可以先手造成大量法术伤害，并在后续通过【转生】大量叠加【业报】触发大日如来的被动效果。布阵时，要考虑保护核心部将。
# 龙宫
龙宫擅于培养龙魂属性，并将龙魂属性附加给龙宫部将来极大的提升作战能力。核心部将东海龙王也可源源不断的召唤神龙，配合龙宫太子能将龙魂收益成倍放大。
